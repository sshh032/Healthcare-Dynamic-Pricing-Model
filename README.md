# 가중 코호트 트렌드를 활용한 헬스케어 보험 프라이싱: 당뇨 신장질환 조기 감지와 이익 곡선 최적화

## 배경
#### 1) 50세 이후 급격한 당뇨병 환자 증가 추세
#### 2) 고령화 사회 진입에 따른 보험사의 유병자 보험 상품 증가 추세
#### 3) 기존 유병자 보험의 획일화된 리스크 평가의 한계점

## 목표: 현재 상태가 아닌 악화속도에 초점을 둔 프라이싱 전략
#### 기존 당뇨 유병자 중 당뇨 신장병증으로의 전환 위험군 선별, 가중 코호트 트렌드를 반영하여 보험료 조정(리스크 최소화)
#### 악화 속도와 추이에 따라 다른 보험료 적용 및 차별화된 관리 제공 가능

## 본론
### STEP 1) 데이터 전처리와 모델링 기법 선택
#### [1] 활용 데이터- 건강보험공단 건강검진데이터 2022~2024 3개년 각 100만 건 샘플링
#### [2] 전처리 전략- 임상적 근거를 통한 이상치 처리 (FPG 40-500, SCR 0.1-15)
#### [3] 가중 코호트 생성- 개인식별자 존재X [성별/연령대/지역기반] 코호트 생성 (유지80 유입20 -> 이동평균 반영)
#### [4] LightGBM 채택- 대규모 정형 데이터의 분류 성능 향상 목적

### STEP 2) 모델 설계
#### [1] XAI기반 '악화속도'의 질병 예측력 증명
##### - SHAP 알고리즘을 이용하여 모델의 판단 근거를 시각화 하고 AI의 설명 불가능성을 해소 후 이해 가능한 산출 기준을 마련
##### - 만성신장질환에 유의미한 영향을 미치는 변수의 중요도 확인
##### - 악화속도가 타겟변수에 미치는 영향력 검증

#### [2] 최적 타켓팅 범위 도출
##### - 고위험군 예측 범위(FP+TP) X 관리비용 + 정상군이라 예측했는데 놓친 범위(FN) X 투석비용
##### = 최적 관리대상 범위 -> 7% 산출

### STEP 3) 모델 구현
#### 우수/보통군(혈당 악화 속도 안정궈) 하위 93%: 고객들에게 월 4.5만원의 보험료 적용과 스마트폰 헬스 앱 리워드 제공(자발적 건강관리 목적)
#### 고위험군(혈당 악화 속도 위험권) 상위 7%: 월 9만원의 보험료 적용과 연속 혈당 측정기 연 2회 제공(고가 보험료에 대한 심리적 보상)
#### 기존 역선택 및 의료비 누수로 인해 연간 1,130억원의 적자 발생에서 프라이싱 밴드 구축으로 연간 184억의 흑자 달성 기대 전환 효과

### STEP 4) 기대효과
#### 보험사: 리스크 방어와 비용절감, 우량고객 확보와 락인효과, 정보 비대칭 차단
#### 고객: 즉각적이고 가시적인 혜택 체감, 악화속도에 맞는 합리적인 보험료 납부
#### 사회: 도덕적 해이 해소, 국가 건강보험 재정 누수 방지
